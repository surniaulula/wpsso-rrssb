WPSSO Core Plugin Required

WPSSO Ridiculously Responsive Social Sharing Buttons (aka WPSSO RRSSB) is an add-on for the WPSSO Core plugin — to install and activate the WPSSO RRSSB add-on, you must already have installed and activated the WPSSO Core plugin.

If you purchased one or more WPSSO RRSSB Premium add-on licenses, you received an email with a unique Authentication ID that will be required to install and update the WPSSO RRSSB Premium add-on.

Install the Add-on

Standard version

  1. Go to the admin section of your WordPress site.
  2. Select the SSO menu item.
  3. Select the Complementary Add-ons settings page.
  4. Scroll down and locate the "WPSSO Ridiculously Responsive Social Sharing Buttons" add-on.
  5. Click the Add-on Details and Install link.

    If the link is named Plugin Details instead (no "Install" in the name) then the add-on is already installed and you can simply activate the add-on from the WordPress Plugins page.

  6. Click the Install Now button in the add-on details window.
  7. Click the Activate Plugin button when WordPress is finished installing the plugin.
  8. The add-on settings page is available under the SSO > Responsive Buttons sub-menu.

Premium version

  1. Install and activate the WPSSO Update Manager add-on (if not already installed and active):
    1. Go to the admin section of your WordPress site.
    2. Select the SSO menu item.
    3. Select the Complementary Add-ons settings page.
    4. Scroll down and locate the "WPSSO Update Manager" add-on.
    5. Click the Add-on Details and Install link.
    6. Click the Install Now button in the add-on details window.
    7. Click the Activate Plugin button when WordPress is done installing the plugin.
  2. Select the SSO menu item.
  3. Select the Licenses settings page.
  4. Scroll down and locate the "WPSSO RRSSB Authentication ID" option.
  5. Enter the unique Authentication ID you received for the WPSSO RRSSB add-on.
  6. Click the "Save All Settings" button at the bottom of the settings page.
  7. Stay in the Licenses settings page.
  8. Scroll down and locate the "WPSSO Ridiculously Responsive Social Sharing Buttons" add-on and its action links.
  9. If the Free / Standard version of WPSSO RRSSB is already installed, click the Add-on Details and Update link:
    1. Then click the Install Update Now button in the add-on details window.
  10. If the WPSSO RRSSB add-on has not been installed yet, click the Add-on Details and Install link:
    1. Then click the Install Now button in the add-on details window.
    2. And click the Activate Plugin button when WordPress is finished installing the add-on.
  11. The add-on settings page is available under the SSO > Responsive Buttons sub-menu.

If you have a WordPress multisite (aka network) setup, please note that the WPSSO RRSSB add-on and the WPSSO Update Manager add-on must be activated on the default site / blog to get Premium version updates. This is a requirement of WordPress, not the WPSSO Core plugin or its add-ons — WordPress uses the default site / blog to manage updates in the network admin interface (which means the default site must be licensed to install updates). You can find additional information on managing multisite licensing here.
