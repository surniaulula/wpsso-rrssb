WPSSO Core Required

WPSSO Ridiculously Responsive Social Sharing Buttons (WPSSO RRSSB) is an add-on for the WPSSO Core plugin - to install and activate this add-on, you must already have installed and activated the WPSSO Core plugin.

Install the Add-on

  1. Go to the admin section of your WordPress site.
  2. Select the SSO menu item.
  3. Select the Plugin Add-ons settings page.
  4. Scroll down and locate the "WPSSO Ridiculously Responsive Social Sharing Buttons" add-on.
  5. Click the Add-on Details and Install link.

    If the link is named Plugin Details instead (no "Install" in the name) then the add-on is already installed and can be activated from the WordPress Plugins page.

  6. Click the Install Now button in the add-on details window.
  7. Click the Activate Plugin button when WordPress is finished installing the plugin.
  8. The add-on settings page is available under the SSO > Responsive Buttons sub-menu.
